# tax

Annual tax and compliance working papers live here.

## Folder Layout

- `tax/<YEAR>/` - Tax return draft package and support for the specified year.

This structure is country-agnostic. Support any jurisdiction (US, Canada, or other) within the year folder.

## What Goes Here

- Draft return workpapers
- Line-mapping JSON or worksheets
- Supporting schedules and reconciliations
- Advisor handoff checklist
- Filing assumptions and open questions

## What Should Not Go Here

- Secrets, credentials, or portal passwords
- Unredacted IDs unless your policy explicitly allows versioning them
- Files unrelated to tax/compliance support

## Conventions

- Keep one folder per tax year.
- Keep assumptions explicit and date-stamped.
- Mark all outputs as draft until professional review.
- Use country-specific skill files for jurisdiction-specific workflows.
