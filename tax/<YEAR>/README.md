# tax/<YEAR>

Tax working papers for year `<YEAR>`.

This folder is country-agnostic and supports any tax jurisdiction. Suggested files vary by country:

**For US filings:**
- `form-1120-draft.md` - Corporate income tax draft
- `form-1120-lines.json` - Line item mappings
- `supporting-schedules.md` - Supporting schedules
- `advisor-handoff-checklist.md` - Advisor review checklist

**For Canada filings:**
- `t2-draft.md` - T2 Corporation Income Tax Return draft
- `t2-lines.json` - Line item mappings
- `supporting-schedules.md` - Supporting schedules
- `advisor-handoff-checklist.md` - Advisor review checklist

**Other jurisdictions:**
Adapt the naming convention to match your local tax forms.

If entity type differs (S-corp, partnership, sole proprietorship, etc.), replace forms and mappings accordingly.
